# 🌤️ Weather App

## Overview
A simple web-based Weather App that provides real-time weather information for selected cities. The application fetches live weather data using the OpenWeatherMap API and displays details such as temperature, humidity, and wind speed. Users can also switch between Celsius (°C) and Fahrenheit (°F).

## Features
- Select a city from a dropdown menu
- Get real-time weather updates
- Display temperature, humidity, and wind speed
- Toggle temperature between Celsius and Fahrenheit
- Clean and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
- OpenWeatherMap API

## How It Works
1. Select a city from the dropdown list.
2. Click the **Get Weather** button.
3. The app sends a request to the OpenWeatherMap API.
4. Weather data is fetched and displayed on the page.
5. Use the **Toggle C/F** button to switch temperature units.

## API Used
OpenWeatherMap API: https://openweathermap.org/api
